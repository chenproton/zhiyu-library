## 一、系统定位与边界

### 1.1 系统定位

面向职业院校的**能力测评认证与毕业设计管理平台**，核心价值是：

- **通用测评资源管理**：构建和维护标准化题库、试卷库，支撑随堂测与在线考试
- **岗位能力测评管理**：基于岗位能力模型，建立岗位认证规则，追踪学生岗位能力达成度
- **测评方式库管理**：统一管理场景任务测评方式的分类、启用状态与关联关系
- **毕业设计管理**：覆盖选题发布、档案跟踪、过程评价到毕业资格查询的全流程
- **学生能力画像管理**：汇聚课程学习、实践场景、毕设评价、档案材料等多维数据，自动生成学生能力画像

### 1.2 系统边界

| 边界项 | 说明/负责方 |
|--------|-----------|
| 用户与组织架构 | 与统一身份认证平台对接，本系统维护用户基础信息与部门关系 |
| 题库/试卷资源 | 本系统独立维护，支持题目类型覆盖单选/多选/判断/填空/简答/问答 |
| 场景任务数据 | 从实践场景平台（zhiyu-scene）同步场景任务信息，用于测评方式关联与结果展示 |
| 在线课堂/智慧课程评价 | 从课程平台同步学生与课堂数据，用于评价评分 |
| 毕业设计流程 | 本系统独立管理选题、档案、评价、查询全链路 |
| 学分与学历认定 | 毕业查询模块与学历认定模块联动，本系统负责展示与状态同步 |
| 能力画像引擎 | 本系统内置画像生成引擎，支持自动计算与手动维护 |

## 三、核心实体与状态

### 3.1 题库（QuestionBank）

题库是题目的容器，支持多维度组织与权限隔离。

#### 基础信息字段

| 字段 | 说明 |
|------|------|
| 题库名称 | 必填 |
| 题库简介 | 文本描述 |
| 题库封面 | 图片上传 |
| 所属批次 | 非必填，关联批次分组 |
| 协作者 | 多选用户/部门，支持共建模式 |
| 版本号 | 系统自动管理 |
| 题目数量 | 只读，自动统计 |
| 所有权类型 | 我的(mine) / 共建(collaborate) / 公共(public) |

#### 题库状态流转

```
草稿(draft)
  ↓ 提交审批
审批中(pending)
  ↓ 审批通过          ↓ 审批驳回
待发布(toPublish)    已驳回(rejected)
  ↓ 发布                ↓ 编辑 → 草稿(draft)
已发布(published)
  ↓ 取消发布
草稿(draft)
```

**各状态可执行操作**：

| 状态 | 可执行操作 |
|------|----------|
| 草稿(draft) | 编辑、删除、提交审批 |
| 未提交(unsubmitted) | 编辑、删除、提交审批 |
| 审批中(pending) | 撤回审批（→未提交） |
| 已驳回(rejected) | 编辑（→草稿）、删除 |
| 待发布(toPublish) | 发布、取消发布（→草稿） |
| 已发布(published) | 取消发布（→草稿） |

**业务规则**：
- R-QB01：草稿库（我的草稿库）为系统默认题库，不可删除
- R-QB02：删除题库时，题库内所有题目同步删除
- R-QB03：已发布题库取消发布后回到草稿状态，可重新编辑

### 3.2 题目（Question）

题目是测评的最小单元，支持 6 种题型。

#### 基础字段

| 字段 | 说明 |
|------|------|
| 所属题库 | 必填，关联到具体题库 |
| 题目类型 | 单选题(single) / 多选题(multiple) / 判断题(judge) / 填空题(fill) / 简答题(short_answer) / 问答题(essay) |
| 题干内容 | 必填，支持文本描述 |
| 选项 | 单选/多选题必填，字符串数组 |
| 正确答案 | 单选为字符串；多选/填空为字符串数组；判断为 true/false；简答/问答为文本 |
| 答案解析 | 非必填，评卷参考 |
| 分值 | 必填，数字 |
| 难度 | 简单(easy) / 中等(medium) / 困难(hard) |
| 知识点 | 多选，关联知识点标签 |
| 来源 | 可标记来源：场景任务/数字课程/试卷/其他来源 |

#### 题目状态流转

题目状态与题库状态独立，支持同样的 6 状态流转规则（草稿/未提交/审批中/已驳回/待发布/已发布）。

**业务规则**：
- R-Q01：题目创建后默认进入草稿库，可手动移动至其他题库
- R-Q02：支持批量移动题目至目标题库
- R-Q03：问答题和简答题为人工评分题型，其余题型支持自动判分

---

### 3.3 试卷（Exam）

试卷是由题目组成的测评集合，支持手动选题与随机抽题两种组卷方式。

#### 基础信息字段

| 字段 | 说明 |
|------|------|
| 试卷名称 | 必填 |
| 试卷简介 | 文本描述 |
| 考试时长 | 必填，单位分钟 |
| 总分 | 只读，自动计算题目分值之和 |
| 所属批次 | 非必填 |
| 协作者 | 多选用户/部门 |
| 所有权类型 | 我的(mine) / 共建(collaborate) / 公共(public) |
| 创建人 | 只读 |

#### 组卷方式

| 方式 | 说明 |
|------|------|
| 手动选题 | 从题库中逐题选择加入试卷，可调整每题分值和顺序 |
| 随机抽题 | 按条件筛选（题库范围/题型/难度/知识点）后自动抽取指定数量题目 |

#### 试卷状态流转

与题库状态流转规则完全一致（草稿/未提交/审批中/已驳回/待发布/已发布）。

**业务规则**：
- R-E01：试卷中的题目以快照形式存储，修改原题库题目不影响已组卷内容
- R-E02：支持调整试卷中题目的顺序和分值
- R-E03：随机抽题组卷时，可预览生成的题目列表

考试是试卷的使用实例，可关联到具体场景或课程。

#### 基础字段

| 字段 | 说明 |
|------|------|

| 使用类型 | 随堂测(quiz) / 在线考试(exam) |

| 结束时间 | 非必填 |
| 参考人数 | 只读统计 |

### 3.5 岗位能力认证规则（CertificationRule）

基于岗位的能力项-能力点结构，定义各岗位的能力认证标准。

#### 实体层级结构

```
岗位（Position）
  └── 能力项（AbilityItem）
        └── 能力点（AbilityPoint）
              ├── 基本信息：名称、描述
              ├── 等级映射：分数区间→等级（继承或自定义）
              ├── 所需掌握度：岗位对该能力点的要求等级
              └── 关联任务：相关场景任务及权重
```

#### 等级映射（LevelMapping）

默认等级映射表：

| 等级 | 分数区间 |
|------|---------|
| 不合格 | 0 - 60 |
| 了解 | 61 - 70 |
| 理解 | 71 - 80 |
| 掌握 | 81 - 85 |
| 熟练 | 86 - 95 |
| 精通 | 96 - 100 |

支持按能力点自定义等级映射，或继承默认映射。

#### 规则状态流转

```
草稿(draft) / 无规则(none) / 未提交(not_submitted)
  ↓ 提交审批
审批中(reviewing)
  ↓ 审批通过          ↓ 审批驳回
待发布(ready)        已驳回(rejected)
  ↓ 发布                ↓ 编辑
已发布(published)
  ↓ 取消发布
待发布(ready)
```

**业务规则**：
- R-CR01：规则来源支持继承（引用系统默认规则）或自定义
- R-CR02：每个能力点可独立配置等级映射和关联任务
- R-CR03：规则发布后即生效，可用于岗位能力测评

### 3.6 测评方式（EvaluationMethod）

测评方式按两级分类结构组织，用于场景任务评价。

#### 实体结构

```
一级测评分类（EvaluationMethodCategory）
  └── 二级测评方式（EvaluationMethod）
        ├── 名称、描述、文档链接
        ├── 启用状态（前台展示开关）
        └── 关联场景任务ID列表
```

**业务规则**：
- R-EM01：测评方式支持启用/禁用，禁用后前台不展示
- R-EM02：每个测评方式可关联多个场景任务
- R-EM03：支持编辑测评方式的说明文档和描述信息

## 四、操作能力清单

### 4.1 通用测评资源管理

#### 4.1.1 题库管理

| 操作 | 触发角色 | 前置条件 | 产出/结果 |
|------|---------|---------|---------|
| 新建题库 | 平台管理员/教师 | — | 题库创建，状态=草稿，版本=v0.1.0 |
| 编辑题库信息 | 创建者/共建人 | 题库状态≠已发布 | 题库基础信息更新 |
| 删除题库 | 创建者 | 题库非草稿库、非已发布 | 题库及下属题目删除 |
| 提交审批 | 创建者 | 状态=草稿/未提交/已驳回 | 状态→审批中 |
| 撤回审批 | 创建者 | 状态=审批中 | 状态→未提交 |
| 发布题库 | 有权限用户 | 状态=待发布 | 状态→已发布 |
| 取消发布 | 有权限用户 | 状态=已发布 | 状态→草稿 |
| 配置协作者 | 创建者 | — | 更新共建用户/部门列表 |
| 调整批次分组 | 创建者 | — | 题库归属批次更新 |
| 导入题库 | 平台管理员 | 文件格式合规 | 批量创建题库/题目 |
| 查看题目列表 | 有权限用户 | 题库存在 | 进入题目管理界面 |
